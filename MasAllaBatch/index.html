Spark Streaming

Procesamiento en Vivo

Es el componente de Spark para procesar flujos de datos en vivo. La implementación original se basaba en DStreams (Discretized Streams), que procesaban los datos en micro-lotes. La API más moderna, Structured Streaming, está construida sobre el motor de Spark SQL.

Spark MLlib

Machine Learning

Es la biblioteca de aprendizaje automático de Spark. Proporciona implementaciones escalables de algoritmos comunes de clasificación, regresión, clustering y filtrado colaborativo, así como herramientas para la construcción de pipelines de ML masivos.
