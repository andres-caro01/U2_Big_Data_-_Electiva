DataFrame

Es una colección distribuida de datos organizada en columnas con nombre, similar a una tabla en una base de datos relacional o un DataFrame en pandas (Python) o R. Los DataFrames se pueden construir a partir de una amplia variedad de fuentes, como archivos estructurados (JSON, CSV, Parquet), tablas de Hive o bases de datos externas. Ofrecen un rico conjunto de operaciones (transformaciones y acciones) y pueden ser manipulados usando expresiones de columna o consultas SQL.

Dataset

Es una extensión de la API de DataFrame que proporciona una interfaz orientada a objetos y con seguridad de tipos (type-safe). Esencialmente, es una colección distribuida de objetos JVM (en Scala y Java). Un DataFrame es, de hecho, un Dataset, donde Row es un objeto genérico y no tipado. Los Datasets combinan la seguridad de tipos de los RDDs con las optimizaciones de rendimiento de los DataFrames.

Carga y Manipulación

La forma más común de interactuar con Spark es a través de la SparkSession, que es el punto de entrada para las APIs estructuradas. A continuación, se muestra un ejemplo simple en PySpark (la API de Python para Spark) de cómo cargar un archivo CSV y realizar algunas transformaciones básicas:

PySpark Code

from pyspark.sql import SparkSession
from pyspark.sql.functions import col, avg

# 1. Crear una SparkSession
spark = SparkSession.builder \
    .appName("EjemploSparkSQL") \
    .getOrCreate()

# 2. Cargar datos desde un archivo CSV
df_ventas = spark.read \
    .option("header", "true") \
    .option("inferSchema", "true") \
    .csv("ventas.csv")

# 3. Mostrar esquema y filas
df_ventas.printSchema()
df_ventas.show(5)

# 4. Transformaciones
df_filtrado = df_ventas.select("producto","cantidad","precio")
df_laptops = df_filtrado.filter(col("producto") == "Laptop")

# 5. Agregación
df_precio_promedio = df_ventas \
    .groupBy("producto") \
    .agg(avg("precio").alias("precio_promedio"))

# 6. Usar SQL directamente
df_ventas.createOrReplaceTempView("vista_ventas")
df_sql = spark.sql("""
  SELECT producto, SUM(cantidad * precio) AS total
  FROM vista_ventas GROUP BY producto
""")

# 7. Detener sesión
spark.stop()
